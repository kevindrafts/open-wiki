## Summary

This short walkthrough shows Koerner setting up a hosted OpenClaw agent through Hostinger, connecting it to Telegram, enabling web search skills, and prompting it to find trending business ideas. The useful pattern is [[ai-agent-operated-businesses]] at the infrastructure level: a lightweight, always-on assistant can monitor opportunities, research trends, and later trigger outreach or implementation tasks.

## Operational playbook

1. Choose a hosted OpenClaw setup so the agent can run while the laptop is closed.
2. Add an Anthropic/OpenAI API key or use a bundled provider key.
3. Connect a messaging surface such as Telegram or WhatsApp.
4. Create a Telegram bot via BotFather and approve the connection.
5. Prompt the agent with a narrow research job and output format.
6. Enable/adjust skills if the default browser automation fails; Koerner installs web-search-related skills and disables browser automation.
7. Refine the prompt by niche, recency, and desired output.
8. Add integrations such as Zapier, Gmail, Twilio, or trained content examples to make the agent proactive and personalized.

## AI/tool workflow

- Hostinger managed OpenClaw installation
